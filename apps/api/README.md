# Save&Serve API

Custom backend scaffold for the Base44 migration.

Current endpoints:

- `GET /health`
- `GET /api/v1/health`
- `GET /openapi.json`
- `GET /api/v1/openapi.json`

Run locally after building:

```bash
npm run api:build
npm run api:start
```

Environment variables are listed in `../../env.example`.
